# `bonsaios setup`

Initialize `~/.bonsaios/bonsaios.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
bonsaios setup
bonsaios setup --workspace ~/.bonsaios/workspace
```

To run the wizard via setup:

```bash
bonsaios setup --wizard
```
