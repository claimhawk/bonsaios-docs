# `bonsaios pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
bonsaios pairing list whatsapp
bonsaios pairing approve whatsapp <code> --notify
```
