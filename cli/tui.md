# `bonsaios tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/tui)

## Examples

```bash
bonsaios tui
bonsaios tui --url ws://127.0.0.1:18789 --token <token>
bonsaios tui --session main --deliver
```
